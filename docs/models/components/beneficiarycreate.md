# BeneficiaryCreate

The beneficiary for transfer on death.

## Example Usage

```typescript
import { BeneficiaryCreate, BeneficiaryCreateRelationType } from "@apexfintechsolutions/ascend-sdk/models/components";

let value: BeneficiaryCreate = {
  beneficiaryPercentage: 100,
  relationType: BeneficiaryCreateRelationType.Spouse,
};
```
