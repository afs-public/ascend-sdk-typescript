# Token

Represents a JSON Web Token for access to the system; See: https://datatracker.ietf.org/doc/html/rfc6749#section-5.1

## Example Usage

```typescript
import { Token } from "@apexfintechsolutions/ascend-sdk/models/components";

let value: Token = {};
```
