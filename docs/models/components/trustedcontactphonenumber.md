# TrustedContactPhoneNumber

The phone number for a party; this value exists on the party record in the context of the account and does not commute to other accounts held by/for the person

## Example Usage

```typescript
import { TrustedContactPhoneNumber } from "@apexfintechsolutions/ascend-sdk/models/components";

let value: TrustedContactPhoneNumber = {};
```
